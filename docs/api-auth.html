Authentication API

Nomchat ID uses email OTP (one-time password) authentication. No passwords are stored.

Flow

1
Send code

POST email → server sends 6-digit code

2
Verify code

POST email + code → server creates session

3
Authenticated

Session cookie set, use /api/auth/me

Endpoints

POST /api/auth/send-code

Send a verification code to the user's email.

Request body

{
  "email": "user@example.com"
}

Response

{
  "success": true,
  "demo_code": null,   // code shown in demo mode (no SMTP)
  "message": "Code sent"
}
Rate limit: 5 requests per 5 minutes per IP.
POST /api/auth/verify

Verify the code and create a session.

Request body

{
  "email": "user@example.com",
  "code": "123456"
}

Response (success)

{
  "success": true,
  "user": { "id": 1, "email": "...", "username": "...", "avatar": "bolt", "is_admin": false },
  "is_new_user": false
}

Response (banned)

HTTP 403
{
  "error": "banned",
  "reason": "Violation of terms",
  "banned_by": "admin@nomchat.id"
}
GET /api/auth/me

Get the currently authenticated user.

Response

{
  "id": 1,
  "email": "user@example.com",
  "username": "player123",
  "avatar": "bolt",
  "is_admin": false,
  "is_dev": false,
  "is_creator": false,
  "is_support": false,
  "is_banned": false,
  "created_at": "2026-01-01T00:00:00",
  "last_login": "2026-04-01T12:00:00"
}
POST /api/auth/logout

Sign out and clear the session.

Response

{ "success": true }

OAuth for external apps

To integrate Nomchat ID into your app, use the OAuth popup flow:

// Open OAuth popup
const popup = window.open(
  'https://nomchat-id.up.railway.app/oauth.html?app_id=your-app&app_name=Your+App',
  'nomchat_auth',
  'width=480,height=640'
);

// Listen for token
window.addEventListener('message', async (event) => {
  if (event.data.type === 'NOMCHAT_AUTH_SUCCESS') {
    const { token } = event.data;
    // Verify token on your server
    const res = await fetch('https://nomchat-id.up.railway.app/api/auth/token/verify', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ token, app_id: 'your-app' })
    });
    const data = await res.json();
    // data.user contains the authenticated user
  }
});
